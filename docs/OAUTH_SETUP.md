# OAuth Setup para Cafe Smart\n\n## Archivos .env\n\n| Archivo | Uso | Variables |\n|---------|-----|----------|\n| `.env` | Web (Vite) | VITE_API_URL, VITE_GOOGLE_CLIENT_ID_WEB |\n| `.env.android` | Android build | VITE_API_URL, VITE_GOOGLE_CLIENT_ID_ANDROID |\n| `.env.example` | Plantilla | Referencia |\n\n## Google Cloud Console - Web Client ID\n\nAgregar a "Origenes autorizados de JavaScript":\n\n| Origen | Plataforma |\n|--------|------------|\n| http://localhost:5173 | Desarrollo web |\n| http://127.0.0.1:5173 | Desarrollo web alterno |\n| http://192.168.X.X:5173 | Dispositivo fisico en red local |\n\nNO agregar 10.0.2.2 - esa IP es solo para API/backend.\n\n## Android Client ID\n\nPara Android nativo se usa SHA-1, package name y redirect URI\n(NO JavaScript Origins).\n\n## Backend URLs por Plataforma\n\n| Plataforma | VITE_API_URL |\n|-------------|--------------|\n| Web | http://localhost:3000 |\n| Emulador Android | http://10.0.2.2:3000 |\n| Dispositivo fisico | http://192.168.X.X:3000 |\n\n## Login Email/Password\n\nFunciona SIN Google OAuth - no afecta si GSI falla.\n
